Service name: Supplier
A basic service exposing a database with a list of parts available for order and the estimated lead time in days for delivery to the client.

URL: http://fastws.qut.edu.au:8080/n5499879SupplierSvc/webresources/entities.part/
WSDL: No WSDL is available, this service is using REST.

A GET HTTP command can be sent to the URL shown above, this will return an XML document containing all parts and corresponding lead times listed in the database.
The URL can also be appended with the specific part description a lead time is required for to retrieve an XML document containing information only about the specified part.

The Supplier service should also have support for placing purchase orders using the PUT command, but due to time constraints and not being able to deploy on the FASTWS server this functionality is at time of submission unsupported.
